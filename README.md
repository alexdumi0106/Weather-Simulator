# Interactive Weather Simulator powered by AI (LLM-based)

## Overview
This project represents my Bachelor’s thesis and reflects my interest in combining meteorology and artificial intelligence into an interactive educational application.
Interactive Weather Simulator is an AI-driven Android application designed to help users visualize and understand how different meteorological parameters influence weather conditions in real time.  
The system combines scientific modeling with natural language explanations generated by a Large Language Model (LLM), creating an educational and engaging experience.
The project was developed in **Android Studio** using **Kotlin** and **Jetpack Compose**, integrating an AI layer that interprets climate data and provides contextual feedback through text generation.

## Main Features
### Real-Time Weather Simulation
Users can adjust parameters such as temperature, humidity, and atmospheric pressure through interactive sliders, immediately visualizing how these affect simulated weather scenarios.
### AI-Generated Explanations
A built-in LLM component generates descriptive insights explaining why certain weather phenomena occur based on the user’s chosen values.
### Dynamic UI with Jetpack Compose
The interface adapts smoothly to parameter changes, ensuring a modern and responsive user experience.
### Data Visualization Layer
Displays the evolution of meteorological variables using dynamic charts and gradient-based graphics.
### Offline & Lightweight Operation
Designed to run efficiently on mobile devices without requiring constant network access.
