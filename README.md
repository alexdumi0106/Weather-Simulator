# Weather Simulator AI

Aplicatie Android care simulează condiții meteo ipotetice bazate pe variabile introduse de utilizator (temperatură, presiune, umiditate).
Un model AI generează explicații despre fenomenele atmosferice rezultate.

## Tehnologii
- Android (Kotlin / Java)
- OpenAI API (LLM)
- Jetpack Compose / XML UI
- GitHub for version control
